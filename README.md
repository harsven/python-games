# Python Games
This repository showcases the evolution of small-scale logic projects from Python exercises to interactive web implementations.

## Overview
These projects began as explorations of programming concepts like **state management**, **collision physics**, and **event-driven logic**. Originally developed in Python using the `simplegui` framework, I have since ported them to native **HTML5 Canvas and JavaScript** to make them playable directly in any browser.

### Interactive Games

#### 1. Pong (Classic Arcade)

## Technical Evolution

### From Python to JavaScript
While the core game logic remains true to the Python originals, the web versions are implemented using the HTML5 Canvas API.

## How to use this Repository
Each folder contains:
* `game_logic.py`: The original Python code (compatible with [CodeSkulptor](https://py3.codeskulptor.org/)).
* `index.html`: The standalone web version used on my website.
